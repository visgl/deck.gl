This is a minimal standalone version of the 3D Plot example
on [deck.gl](http://deck.gl) website.

### Usage

From the deck.gl repository root, install workspace dependencies and run this example from its directory.

```bash
# From the deck.gl repository root
yarn install

# From this example directory
yarn start
```

### Data format

Check out the [documentation of PlotLayer](./plot-layer/README.md).
