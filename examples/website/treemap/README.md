This is a standalone version of the ClippingTextExtension example on [deck.gl](http://deck.gl) website.

### Usage

From the deck.gl repository root, install workspace dependencies and run this example from its directory.

```bash
# From the deck.gl repository root
yarn install

# From this example directory
yarn start
```

### Data format

Sample data is from [D3 Treemap demo](https://observablehq.com/@d3/treemap/2).
